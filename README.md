# Portifólio
Por meio do desafio 'Start sua Carreira' da Kenzie Academy, foi desenvolvido um currículo/portifólio, utilizando HTML e CSS. Foi implementado a função 'dark-mode'.

## Estrutura do diretório:

 - Portifolio/
	 - index.html
	 - style/
		 - style.css
	 - layout/
		- desktop
		- desktop-dark
		- mobile
		- mobile-dark 
	- img/
		- minha-foto
	  	- svg/
			- github-big-logo
			- linkedin
	
## Especificações

-   Fonte utilizada:  `'Open Sans', sans-serif;`
-   Cores utilizadas:
    -   Azul:  `#3d69f7`;
    -   Branco:  `#fff`;
    -   Dourado:  `#bd8d0e`;
    -  Cinza escuro: `#252525`;
    -   Preto `#000`;
